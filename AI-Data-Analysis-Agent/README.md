
# AI Data Analysis Agent

This project demonstrates a simple AI-style data analysis assistant for Data Analyst portfolios.

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Automatic insights generation
- Modular Python project structure

## Tech Stack
Python, Pandas, Matplotlib, Seaborn

## Project Structure

AI-Data-Analysis-Agent
│
├── data
│   └── sales_data.csv
├── notebooks
├── src
│   ├── data_cleaning.py
│   ├── eda.py
│   └── ai_insights.py
├── app.py
├── requirements.txt
└── README.md

## Run Project

pip install -r requirements.txt

python app.py
