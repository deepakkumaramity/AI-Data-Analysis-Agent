# AI Data Analysis Agent

## Project Overview
AI Data Analysis Agent is a Python-based project that automatically analyzes datasets and generates insights.  
This tool performs data cleaning, exploratory data analysis (EDA), and generates useful insights from the dataset.

This project demonstrates how automation and AI-style logic can help data analysts understand data faster.

---

## Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Data Visualization
- Automated Insight Generation

---

## Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset

The project uses a sample sales dataset that contains information such as:

## Installation

Clone the repository

git clone https://github.com/yourusername/AI-Data-Analysis-Agent.git

Move to project folder

cd AI-Data-Analysis-Agent

Install required libraries

pip install -r requirements.txt

---

## Run the Project

python app.py

The script will:

- Load dataset
- Perform data cleaning
- Analyze the dataset
- Generate insights
